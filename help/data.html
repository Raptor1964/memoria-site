🗄️ Gestione Database

In questa pagina puoi gestire la sicurezza e l'integrità dei tuoi archivi con le seguenti opzioni:

💾 BACKUP: Salva una copia di sicurezza del database nello spazio riservato dell’app sul telefono. Utile per creare un punto di ripristino rapido.

⏪ RIPRISTINA: Elimina il database attuale e carica l’ultimo backup salvato localmente.

Attenzione: i dati inseriti dopo l'ultimo backup andranno persi.

📤 CONDIVIDI DATI: Esporta una copia crittografata sulla memoria esterna o verso servizi cloud per portarli su un altro dispositivo.

📥 IMPORTA DATI: Recupera e carica un file di backup precedentemente salvato nella memoria esterna dello smartphone.

☁️ GOOGLE DRIVE: Accedi alla configurazione per sincronizzare, salvare o ripristinare i tuoi backup direttamente sul cloud di Google.

🔑 NUOVA CHIAVE: Cambia la chiave di crittografia AES-256 del database.

Questa è la serratura digitale che protegge i tuoi dati: assicurati di non dimenticare la nuova chiave!
