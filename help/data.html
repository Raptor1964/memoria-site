🗄️ Database Management

On this page, you can manage the security and integrity of your archives with the following options:

💾 BACKUP: Saves a security copy of the database in the app's reserved space on your phone. Useful for creating a quick restore point.

⏪ RESTORE: Deletes the current database and loads the last locally saved backup.

Warning: Any data entered after the last backup will be lost.

📤 SHARE DATA: Exports an encrypted copy to external storage or cloud services to move it to another device.

📥 IMPORT DATA: Retrieves and loads a backup file previously saved on the smartphone's external memory.

☁️ GOOGLE DRIVE: Access the configuration to sync, save, or restore your backups directly to the Google cloud.

🔑 NEW KEY: Changes the AES-256 encryption key of the database.

This is the digital lock that protects your data: make sure you do not forget the new key!
