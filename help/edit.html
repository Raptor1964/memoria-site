✏️ Edit Service

All fields on this page are editable to reflect changes in your accounts:

📝 EDITABLE FIELDS: You can freely update the Service Name, Username, Website, and Notes.

🔐 PASSWORD UPDATE: If you modify the password, Memoria will automatically update the "Modified Date" in the service details, helping you remember how recent your access key is.

🖼️ ICON MANAGEMENT: If you change the Website address and save, the app will attempt to download the corresponding new icon.

If the download is not successful, the default Memoria icon will be used.

🔘 'ICON' BUTTON: This button allows you to manually choose which image to display, toggling between the icon downloaded from the web and the app's standard icon.

💾 SAVE: Tap 'Save' to make the changes effective in the database.
